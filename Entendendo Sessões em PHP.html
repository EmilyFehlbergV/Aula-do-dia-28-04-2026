Gerenciamento de Sessões em PHP

Como manter o estado do usuário entre diferentes páginas

1. Criando a Sessão (Login)

Para iniciar qualquer manipulação de sessão, usamos o session_start(). Ele cria um ID único no servidor e envia um cookie para o navegador do usuário.

<?php
session_start(); // Essencial: deve ser a primeira linha!

// Guardamos informações na "gaveta" global do servidor
$_SESSION["usuario"] = "Gemini";
$_SESSION["id"] = 123;
$_SESSION["logado"] = true;

echo "Sessão iniciada e variáveis gravadas!";
?>

2. Recuperando Dados (Perfil)

Em outra página, o PHP reconhece o usuário pelo ID do cookie e reidrata a superglobal $_SESSION com os dados que salvamos antes.

<?php
session_start(); // Conecta com a sessão existente

if (isset($_SESSION["usuario"])) {
    echo "Bem-vindo, " . $_SESSION["usuario"] . "!";
    echo "Seu ID é: " . $_SESSION["id"];
} else {
    echo "Você não está logado.";
}
?>

3. Encerrando a Sessão (Logout)

Para sair do sistema, primeiro limpamos as variáveis e depois destruímos o arquivo de sessão no servidor.

<?php
session_start();

session_unset(); // Limpa as variáveis da memória
session_destroy(); // Destrói o arquivo da sessão

echo "Você saiu do sistema.";
?>

Dica: O session_start() deve ser chamado em todos os arquivos onde você deseja acessar os dados do usuário.
